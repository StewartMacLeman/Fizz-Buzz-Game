Fizz / Buzz

  1. Select the type of game - "Counter Game" or "Random Game". Then click Begin!
    • Select "None" for numbers that can't be equally divided by 3 or 5.
    • Select "Fizz" for numbers that can be equally divided by 3.
    • Select "Buzz" for numbers that can be equally divided by 5.
    • Select "Fizz-Buzz" for numbers that can be equally divided by 3 or 5.
  2. Click the "Reset Games" button to start again.
